# Surprize Muzicale – Comenzi de melodii personalizate

Acesta este proiectul oficial pentru surprizemuzicale.ro, un site unde clienții pot comanda melodii personalizate create special pentru ei.

## Structura proiectului

surprizemuzicale/
│
├── frontend/
└── server/

## Funcționalități

- Comandă melodii personalizate
- Plată cu cardul prin Stripe
- Panou Admin
- Webhook Stripe
- MongoDB
